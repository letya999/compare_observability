# Security Policy

## Supported Versions

Currently, only the latest version of this project is supported.

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not open a public issue. Instead, please report it via email to the maintainers.

We will acknowledge your report within 48 hours and provide a timeline for a fix if necessary.

Please include the following information in your report:
- Type of issue (e.g., buffer overflow, SQL injection, cross-site scripting, etc.)
- Full paths to source files related to the manifestation of the issue
- The location of the affected code
- A proof-of-concept or exploit code if possible
- Impact of the vulnerability

Thank you for helping keep this project secure!
